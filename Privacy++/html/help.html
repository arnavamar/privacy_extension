What is Privacy++?

Privacy++ is a browser extension which has four main functions. HTTPS everywhere, Passoword Manager, JavaScript blocker and Link reveal

What is HTTPS everywhere?

HTTPS everywhere function block all websites that are not using HTTPS protocol.

What is Password Manager?

Password Manager is a function that gives the user the opportunity to store passwords in an encrypted form.

What is JavaScript blocker?

JavaScript blocker blocks JavaScript in every website to ensure security for the user while using the browser.

What is Link Unshortner?

Link Unshortner is a function that enables the user to see where certain short links go to.

I forgot my master password, Can I recover it?

In case of loosing the master password there is nothing you can do, all your stored passwords would be lost. The reason why Privacy++ works in this way is to achieve the maximum security.
